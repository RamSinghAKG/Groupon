# Groupon
Project Setup steps:
1) git clone https://github.com/RamSinghAKG/Groupon.git
2) cd server
3) npm install
4) nom start   //http://localhost:9000 
5) cd client
6) npm install
7) npm start //open browser http://localhost:3000
